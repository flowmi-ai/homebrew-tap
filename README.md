# Homebrew Tap for Flowmi CLI

This is the official [Homebrew](https://brew.sh) tap for [Flowmi CLI](https://github.com/flowmi-ai/flowmi).

## Install

```sh
brew install flowmi-ai/tap/flowmi
```

## Update

```sh
brew upgrade flowmi
```

## Note

The formula in this repository is **automatically generated and updated by [GoReleaser](https://goreleaser.com/)** on each release of [flowmi-ai/flowmi](https://github.com/flowmi-ai/flowmi). Do not edit `Formula/flowmi.rb` manually — your changes will be overwritten on the next release.
